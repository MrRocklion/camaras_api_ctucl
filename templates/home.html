Informacion

La API actual fue desarrollada con el propósito de facilitar la configuración y el acceso a los datos del sistema de monitoreo de buses. Entre las tecnologías empleadas para su desarrollo se encuentran: Flask: Framework de Python utilizado para gestionar solicitudes HTTP y realizar consultas directas a la base de datos, permitiendo la obtención eficiente de información. SQLite: Base de datos ligera empleada para almacenar datos relacionados con las transacciones y el sistema de conteo. HTML y CSS: Tecnologías utilizadas para la presentación y diseño de la interfaz de usuario. Este conjunto de herramientas garantiza un sistema robusto y eficiente para la gestión del monitoreo de buses.
